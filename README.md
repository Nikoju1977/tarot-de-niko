# Jodorowsky Tarot Oracle Application

## Description
A tarot card reading application inspired by the Jodorowsky Tarot. The application seeks to provide insights and a deeper understanding through tarot card readings.

## Features
- Interactive tarot card readings
- Customizable tarot spreads
- History of past readings
- Detailed card descriptions and meanings
- Multi-language support

## Improvements
- Optimized user interface for a smoother experience
- Implemented additional tarot spreads based on user feedback
- Enhanced performance and loading times
- Updated card meanings based on the latest research

## Installation Instructions
To install the application:
1. Clone the repository: `git clone https://github.com/Nikoju1977/tarot-de-niko.git`
2. Navigate to the project directory: `cd tarot-de-niko`
3. Install dependencies: `npm install`
4. Start the application: `npm start`

## Usage Guide
- Launch the application and select a tarot spread.
- Choose your cards by clicking on them.
- Click "Read" to interpret the cards based on their positions.
